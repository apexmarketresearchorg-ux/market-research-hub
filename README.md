# 🚀 Market Research Hub - Full Stack Application

A complete, production-ready Market Research Hub built with Next.js, Node.js/Express, and MongoDB.

## ✨ Features

✅ **Beautiful Modern UI** - Dark theme with Tailwind CSS & Framer Motion
✅ **Admin Dashboard** - Easy content management without coding
✅ **Drag & Drop Uploads** - Upload documents to cloud storage
✅ **Cloud Storage Integration** - AWS S3 integration
✅ **Responsive Design** - Mobile-first approach
✅ **Real-time Updates** - React Query data management
✅ **SEO Optimized** - Next.js built-in SEO
✅ **Authentication** - JWT-based auth system
✅ **Docker Ready** - One-command deployment
✅ **TypeScript** - Type-safe codebase

## 📋 Tech Stack

### Frontend
- Next.js 14 (React 18)
- TypeScript
- Tailwind CSS
- Framer Motion
- React Query
- Axios

### Backend
- Node.js + Express
- TypeScript
- MongoDB
- JWT Authentication
- AWS S3
- Multer (File uploads)
